# Checklist de cumplimiento — Perfil ADVANCED (Analista de Integración)

## 1) Perfil / definición del rol
- [ ] Se describe que el analista de integración **avanzado** tiene experiencia en el **diseño, desarrollo y mantenimiento** de arquitecturas de integración complejas.
- [ ] Se evidencia conocimiento sólido en tecnologías de integración: **ESB, EAI, REST, SOAP, mensajería, API Gateway y plataformas de integración**.
- [ ] Se evidencia experiencia en **implementación y administración de microservicios**.
- [ ] Se evidencia experiencia en **desarrollo de APIs** e integración con sistemas heterogéneos.
- [ ] Se contempla integración con **SaaS** y **productos de terceros en la nube**.
