# Checklist de cumplimiento — Perfil JUNIOR (Analista de Integración)

## 1) Perfil / definición del rol
- [ ] Se describe que el analista de integración **junior** se especializa en la **conexión y comunicación** de sistemas.
- [ ] Se menciona que su rol implica **diseño, desarrollo y mantenimiento** de soluciones de integración.
- [ ] Se listan tecnologías típicas: **ESB, EAI, REST, SOAP y mensajería**.

---

## 2) Fundamentos: SOA – API – Microservicios
- [ ] Explica la **creación y mantenimiento** de **APIs, servicios o microservicios**.
- [ ] Define qué es la **documentación de una API** y por qué es **crucial**.
- [ ] Explica cómo se implementa un **patrón de bajo acoplamiento**.
- [ ] Describe el **ciclo de vida de las APIs**.

## 3) Patrones
